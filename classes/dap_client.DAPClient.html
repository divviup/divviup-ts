Class DAPClient<Spec, Measurement>

A client for interacting with DAP servers, as specified by draft-ietf-ppm-dap-00. Instances of this class contain all of the necessary functionality to generate a privacy-preserving measurement report for the provided ClientVdaf, such as an implementation of Prio3, as specified by draft-irtf-cfrg-vdaf-00.
